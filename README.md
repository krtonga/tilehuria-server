Server for running [TileHuria](https://github.com/ivangayton/tilehuria).


###TOOLS FOR DEVELOPMENT

To run on localhost: 
```commandline
virtualenv venv
export FLASK_APP=tilehuriaserver.py
python tilehuriaserver.py
```
